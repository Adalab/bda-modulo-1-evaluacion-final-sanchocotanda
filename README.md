# bda-modulo-1-evaluacion-final-sanchocotanda
bda-modulo-1-evaluacion-final-sanchocotanda created by GitHub Classroom

Proyecto basaddo en el sistema de gestión de una tienda, en este caso se trata de una paquetería.
Autora: Maria Sancho

A partir de una lista vacía, este proyecto permite:
- Añadir productos
- Ver inventario
- Buscar productos
- Actualizar stock
- Eliminar productos
- Calcular valor total
- Realizar compras

Todo ello poniendo en práctica:

- **Listas** para almacenar el inventario.
- **Diccionarios** para representar cada producto.

**Funciones**
- Uso de funciones modulares para organizar el código.
- Empleo de `return` para devolver resultados.
- Uso de `None` como valor por defecto o indicador de ausencia.

**Control de flujo**
- Bucles **`for`** para recorrer el inventario.
- Bucles **`while`** para menús interactivos y procesos repetitivos.
- Condicionales **`if`**, **`elif`**, **`else`** para tomar decisiones.
- Instrucción **`break`** para salir de bucles cuando es necesario.

**Valores lógicos**
- Uso de **`True`** y **`False`** en validaciones y condiciones.

**Interacción con el usuario**
- Empleo de **`input()`** para recibir datos y gestionar el flujo de compra.
